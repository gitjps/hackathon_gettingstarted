Getting Started with IBM Cloud at Hackathons

Welcome!

On this page I'd like to share some information that you can use to prepare for the upcoming hackathon.

The IBM Cloud is a comprehensive Plattform-as-a-Service solution that allows to rapidly develop applications.

  • It consists of more than 170 services, such databases, IoT, artificial intelligence etc.. This blog provides some general recommendations how to prepare for hackathons.
  • Multiple runtimes can be used (e.g. Node.js, Java, Python). If you have limited coding experiences you might want to consider using Node-RED , which is a graphical wiring tool, accelerating the coding.
  • Chatbots can be used in a variety of use cases and the Watson Assistant services is the perfect tool to create them as described in this demo and these instructions for developers.

Questions? Please email me! Looking forward to see you at the hackathon and have fun!

Peter Schleinitz, IBM
